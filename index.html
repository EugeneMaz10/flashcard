JS / jQuery Flashcards

JS terms:

  1. JavaScript
    • JavaScript is a high-level, dynamic, untyped, and interpreted programming language.
  2. Modulo
    • In computing, the modulo operation finds the remainder after division of one number by another.
  3. Operators
    • JavaScript has both binary and unary operators, and one special ternary operator, the conditional operator. A binary operator requires two operands, one before the operator and one after the operator.
  4. NaN
    • NaN is a property of the global object. The initial value of NaN is Not-A-Number.
  5. Variable naming conventions
    • In computer programming, a naming convention is a set of rules for choosing the character sequence to be used for identifiers which denote variables, types, functions, and other entities in source code and documentation.
  6. +=
    • The addition assignment operator (+=) adds a value to a variable.
  7. Function
    • A JavaScript function is defined with the function keyword, followed by a name, followed by parentheses ().
  8. Method
    • JavaScript methods are the actions that can be performed on objects. A JavaScript method is a property containing a function definition.
  9. Argument to a method
    • JavaScript functions have a built-in object called the arguments object. The argument object contains an array of the arguments used when the function was called (invoked).
  10. String
    • A string is traditionally a sequence of characters, either as a literal constant or as some kind of variable. The latter may allow its elements to be mutated and the length changed, or it may be fixed (after creation).
  11. Chaining methods
    • Method chaining is a technique that can be used to simplify code in scenarios that involve calling multiple functions on the same object consecutively.
  12. Escape
    • The backslash itself is used as the escape character. Therefore, you cannot directly type one in your script. If you want to write a backslash, you must type two of them together (\\).
  13. Alert
    • The alert() method displays an alert box with a specified message and an OK button.
  14. Commenting code
    • Single line comments start with //. Any text between // and the end of the line, will be ignored by JavaScript (will not be executed).
    • Multi-line comments start with /* and end with */. Any text between /* and */ will be ignored by JavaScript.
  15. Booleans
    • Boolean data type is a data type, having two values (usually denoted true and false), intended to represent the truth values of logic and Boolean algebra.
  16. Undefined
    • The undefined property indicates that a variable has not been assigned a value.
  17. Parameter
    • Function parameters are the names listed in the function definition. Function arguments are the real values passed to (and received by) the function.
  18. Return
    • The return statement stops the execution of a function and returns a value from that function.
